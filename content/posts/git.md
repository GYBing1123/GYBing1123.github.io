# Git 简介
Git 是一种分布式的版本控制系统
- 分布式：可以没有中心服务器，每个终端可以独立运行。
- 版本控制系统：记录文件的变化情况。

# Git的最小配置
在使用 git 之前，要对 git 进行最基本的配置。
```bash
# 配置用户名
git config --global user.name "username"

# 配置用户邮箱
git config --global user.email "useremail@"
```
